## 📊 HOW TO USE EACH FEATURE

### Getting Started
1. **Sign In** - Click "Continue with Google" on login page
2. **Access Dashboard** - Auto-directed after login
3. **Explore Features** - Use sidebar navigation

### Creating Your First Assignment
1. Go to **Assignments** page
2. Click **"+ New Task"** button (yellow)
3. Fill in:
   - **Title** - Assignment name
   - **Description** - Details
   - **Priority** - Urgency level
   - **Due Date** - When it's due
4. Click **"Create Assignment"**
5. Watch it appear in the **To Do** column

### Managing Study Goals
1. Go to **Study Goals** page
2. Click **"+ New Study Goal"** button
3. Enter goal name (e.g., "Daily reading")
4. Toggle to **"Active"**
5. Mark complete each day
6. Watch your streak grow!

### Tracking Progress
1. Visit **Dashboard** for quick stats
2. Check **Calendar** for upcoming assignments
3. View **Analytics** for detailed performance
4. Monitor **Degree Progress** toward graduation

### Viewing Courses
1. Go to **Courses** section
2. See all your enrolled courses
3. View progress percentage
4. Check instructor and grade
5. See next assignment deadline

### Customizing Settings
1. Go to **Settings**
2. Update **Account Settings** - Name, email preferences
3. Set **Learning Preferences** - Study notifications
4. Configure **Analytics** - What data to track
5. Click **"Save Settings"**

## 🐛 COMMON ISSUES & SOLUTIONS

### Issue: Tasks not showing up
**Solution**: 
- Refresh page (Ctrl+R)
- Check you're logged in
- Verify no console errors (F12)
- Wait a few seconds for sync

### Issue: Can't create assignment
**Solution**:
- Fill all required fields
- Check due date format
- Verify Firebase connection
- Try again if internet is slow

### Issue: Slow performance
**Solution**:
- Close browser tabs
- Clear browser cache (Ctrl+Shift+Delete)
- Refresh page
- Check internet speed

### Issue: Logged out unexpectedly
**Solution**:
- Check internet connection
- Firebase session may have expired
- Sign in again
- This is normal after 24+ hours idle
